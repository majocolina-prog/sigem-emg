# SiGEM-EMG

Prototipo inicial de una Historia Clínica Electrónica para un Servicio de
Emergentología.

## Funciones incluidas

- Inicio de sesión demostrativo y permisos por rol.
- Tablero de pacientes.
- Admisión y datos del episodio de guardia.
- Evaluación inicial y evoluciones médicas.
- Indicaciones médicas imprimibles.
- Hoja de enfermería, medicación y balance hídrico.
- Estudios complementarios e interconsultas.
- Alta, internación, derivación, observación u óbito.
- Consumo semanal de medicación para reposición.
- Reportes y auditoría básica.

## Cómo probarlo

Abrir `index.html` en un navegador. La contraseña demostrativa es `demo`.

## Estado del proyecto

Esta es una maqueta funcional para validar pantallas y circuitos de trabajo. No
debe utilizarse todavía con datos reales de pacientes. Antes de usarla en una
institución requiere una base de datos segura, autenticación real, copias de
respaldo, cifrado, controles de acceso y revisión legal y sanitaria.
